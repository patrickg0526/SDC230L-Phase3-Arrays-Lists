# SDC230L - Phase 3: Arrays and Lists
Author: Patrick Gonzalez
Course: SDC230L

## Project Structure
- src/CalculatorApp.java - Main application entry point
- src/DisplayManager.java - All console output and menus  
- src/InputManager.java - User input with validation
- src/SingleMemory.java - Single numeric value memory
- src/CollectionMemory.java - ArrayList collection memory (up to 10 integers)

## Compile and Run
cd src
javac *.java
java CalculatorApp
